<span data-ttu-id="ce931-127">支援服務的工程師會根據其角色授與安全性群組的成員資格。</span><span class="sxs-lookup"><span data-stu-id="ce931-127">Engineers supporting a service are granted membership to security groups based on their role.</span></span> <span data-ttu-id="ce931-128">安全性群組中的成員資格不會授與任何許可權存取權。</span><span class="sxs-lookup"><span data-stu-id="ce931-128">Membership in a security group does not grant any privileged access.</span></span> <span data-ttu-id="ce931-129">相反地，安全性群組可讓工程師在必要時，使用密碼箱來要求 JIT 提升以支援系統。</span><span class="sxs-lookup"><span data-stu-id="ce931-129">Instead, security groups allow engineers to use Lockbox to request JIT elevation when required for supporting the system.</span></span> <span data-ttu-id="ce931-130">工程師可以採取的特定 JIT 要求是由其安全性群組成員資格所限制。</span><span class="sxs-lookup"><span data-stu-id="ce931-130">The specific JIT requests an engineer can make are limited by their security group memberships.</span></span>

## <a name="how-does-microsoft-365-handle-remote-access-to-production-systems"></a><span data-ttu-id="ce931-131">Microsoft 365 如何處理生產系統的遠端存取？</span><span class="sxs-lookup"><span data-stu-id="ce931-131">How does Microsoft 365 handle remote access to production systems?</span></span>
